Fourth test Front project made with free Figma template. Just for learn front UI.

You can see [demo!](https://sviat-o-slav.github.io/furniking/)

## Requiremetns:
- npm 8.5.0
- node 16.14.2
## Develop deploy
Install all dependencies:
```
npm install
```
Build all project:
```
gulp build
```
Run Browser-Sync watch mode server:
```
gulp dev
```
